## 🚀 Features

* <b>Massive Library</b>: Over 800+ drugs pre-mapped from common analgesics to critical care agents like Dexmedetomidine and Sugammadex
* <b>Smart Insertion</b>: Automatically appends brand names in parentheses, e.g., <i>Acetaminophen (Tylenol)</i>
* <b>Exclusion Logic</b>: Blacklist specific tags (like <code>NoBrand</code>) or entire deck hierarchies to keep your non-med cards clean
* <b>Custom Mapping</b>: Add your own drug pairs via a streamlined UI directly in Anki's Tools menu

## 🛠 Installation

1. Download the `.ankiaddon` file from the [Releases](#) page
2. Open Anki &rarr; Tools &rarr; Add-ons
3. Drag and drop the file into the window and restart Anki
