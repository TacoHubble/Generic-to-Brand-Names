# Generic to Brand Names (Anki Add-on)

A pharmacology-focused Anki add-on designed to bridge the gap between generic drug names and their commercial counterparts. Perfect for medical students and clinicians who need to recognize both naming conventions in real-time.

## 🚀 Features
* **Massive Library**: Over 800+ drugs pre-mapped from common analgesics to critical care agents like Dexmedetomidine and Sugammadex.
* **Smart Insertion**: Automatically appends brand names in parentheses, e.g., *Acetaminophen (Tylenol)*.
* **Exclusion Logic**: blacklist specific tags (like `NoBrand`) or entire deck hierarchies to keep your non-med cards clean.
* **Custom Mapping**: Add your own drug pairs via a streamlined UI directly in Anki's Tools menu.

## 🛠 Installation
1. Download the `.ankiaddon` file from the [Releases](link-to-your-release-later) page.
2. Open Anki -> Tools -> Add-ons.
3. Drag and drop the file into the window and restart Anki.

## ⚙️ Configuration
Access settings via **Tools -> Generic to Brand Names**. 
* **Case Sensitivity**: Toggle strict matching.
* **Delimiters**: Customize how multiple brand names are separated (default is `; `).
* **Exclusions**: Manage your excluded decks and tags through the dedicated UI.
